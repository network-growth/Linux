## 오늘 배운 명령어 요약
- `pwd`: Print Working Directory (내 위치 찾기)
- `ls`: List (목록 보기)
- `mkdir`: Make Directory (폴더 만들기)
- `cat`: Concatenate (파일 내용 보기)
