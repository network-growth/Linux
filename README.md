## 1일차: 리눅스 서버 설치 및 기초 구성 (2026-02-03)

오늘은 네트워크 엔지니어가 되기 위한 첫걸음으로 내 컴퓨터에 가상 서버를 구축했습니다.

### 실습 환경
- **가상화 도구**: Oracle VirtualBox 7.2.6a
- **운영체제**: Ubuntu Server 24.04.3-live-server-amd64

### 📸 주요 설치 과정
1. **VirtualBox 설정**: 네트워킹과 USB 지원 옵션을 모두 포함해 설치했습니다.
2. **가상 머신 생성**: 이름은 `one stap`으로 정했고, D 드라이브에 안전하게 저장했습니다.
3. **OS 설치**: 수동 설치 모드로 진행하여 계정 설정과 SSH 설정을 직접 마쳤습니다.

### 💡 오늘 해결한 문제 (Troubleshooting)
- 처음 사용해보는 Oracle virtual box 를 사용해보니 어려움이 있었지만 AI 제미나이를 활용하여 나아갔습니다
- 가상 머신 안으로 마우스가 들어가서 안 나올 때 **오른쪽 Ctrl 키**를 누르면 탈출할 수 있다는 것을 배웠습니다.
- 영어로 된 설치 화면이었지만, 차근차근 읽으면서 프록시나 미러 서버 설정을 완료했습니다.
